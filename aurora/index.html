מובייל · אותה רשת Wi‑Fi

הטלפון והמחשב חייבים להיות על אותה רשת אלחוטית (בית / נקודה חמה מהמחשב). לא יעבוד מסלולר בלבד כשהמחשב בבית והטלפון בנסיעה ללא VPN או מנהרה.

התקנה למשתמש חדש · מחשב + מובייל

מטרה: שרת מלא במחשב → Aurora במובייל באותה Wi‑Fi → חיפוש YouTube → שמירה במכשיר → האזנה גם בלי רשת הבית.

עם לחיצה על ״התקנה למחשב חדש״: מועתקת פקודת PowerShell (תמיד עובדת אחרי Python). אם ב-GitHub יש Release עם המתקין — גם מתחילה הורדת ה־exe. אם לא — פתחי PowerShell והדביקי.

שלב א׳ — מתקין Windows (מומלץ)

  1. התקיני Python 3.10+ עם Add python.exe to PATH (אם חסר): ב־PowerShell או CMD כמנהל — winget install Python.Python.3.13 --scope machine (לכל המחשב). בלי מנהל: winget install Python.Python.3.13 --scope user. אחרי ההתקנה — סגרי את כל חלונות PowerShell/CMD (או הפעילי מחדש את המחשב) ואז הריצי שוב את המתקין.
  2. הורידי את הקובץ CarPlayerAurora-Setup.exe, הריצי אותו ואשרי את אשף ההתקנה של Windows. בסוף ייפתחו חלון שרת (השאירי פתוח) ודפדפן עם Aurora.
  3. אחרי הפעלה מחדש של המחשב — ההפעלה לא בדפדפן בלבד: צריך שוב להריץ את השרת.
    • שולחן עבודה (אם סימנת באשף): קיצור Car Player server — לחיצה כפולה פותחת את השרת; Aurora פותחת את הדפדפן (רק כשהשרת כבר רץ).
    • תפריט התחלה → תיקייה Car Player → Start Car Player server.
    • תיקיית הקבצים (למתקדמים): %LOCALAPPDATA%\CarPlayer-Aurora — שם start-server-lan.bat.
    • שורת משימות: אין ב-Windows אישור אחד מהאפליקציה ש״ינעץ אוטומטית״ קיצור — אחרי שהשרת רץ, לחצי ימין על האייקון של חלון השרת בשורת המשימות ובחרי הצמד לשורת המשימות (או גררו מהשולחן עבודה).
  4. במובייל (אותה Wi‑Fi): כפתור חיבור מובייל באותה רשת או http://<IP-של-המחשב>:5600/aurora/index.html — אחרי ~3 שניות תופיע התראה להתקנה למסך הבית.
  5. חיפוש → ניגון → אופליין → העתק למכשיר. אחר כך מצב רק במכשיר לניגון בלי השרת.
הורד CarPlayerAurora-Setup.exe עמוד גרסאות

אם לחיצה כפולה על ה־exe לא מראה חלון: לחצי ימין על הקובץ → מאפיינים → סמני שחרור חסימה אם מופיע; אם נפתח חלון כחול של Windows — מידע נוסף → הרץ בכל זאת. לפעמים חלון האשף נפתח מאחורי הדפדפן — בדקי בשורת המשימות. לוג לאבחון: ב־cmd בתיקיית ההורדות הריצי CarPlayerAurora-Setup.exe /LOG=%TEMP%\aurora-inno.log ואז פתחי את קובץ הלוג ב־%TEMP%.

אם ההורדה מחזירה 404: ב־car-player-aurora → Actions → Build Windows installer → Run workflow (פעם אחת). אחרי בנייה מוצלחת הקישור למעלה עובד. עד אז: שלב ב׳.

שלב ב׳ — PowerShell (אם אין .exe)

  1. פתחי PowerShell, לחצי העתק פקודת התקנה אוטומטית והדביקי → Enter. (מוריד ל־%USERPROFILE%\CarPlayer-Aurora, מתקין חבילות, מפעיל שרת.)
צפי בסקריפט

שלב ג׳ — התקנה ידנית (ZIP)

  1. Download ZIP → חילוץ לתיקייה.
  2. פעם ראשונה בתיקייה: pip install -r requirements.txt
  3. כל הפעלה: לחיצה כפולה על start-server-lan.bat (לא http.server).
  4. מחשב: http://127.0.0.1:5600/aurora/index.html · מובייל באותה רשת: http://<IP>:5600/aurora/index.html

מומלץ להתקין למסך הבית כדי לפתוח מהר כמו אפליקציה, עם חוויה מלאה במסך מלא.

קישור כמו אתר · HTTPS

כדי לקבל כתובת https://… שנפתחת בטלפון מכל מקום (כמו אתר), משתמשים ב־Cloudflare Quick Tunnel יחד עם השרת המקומי.

  1. התקיני פעם אחת את cloudflared (חינמי):
    PowerShell כמנהל: winget install Cloudflare.cloudflared
    או הורדה מ־GitHub — cloudflared
  2. סגרי חלונות שרת ישנים, ואז לחצי כפול על הקובץ בתיקיית הפרויקט:
    start-public-https.bat
  3. ייפתחו שני חלונות: אחד לשרת, אחד למנהרה. בחלון המנהרה תופיע שורה עם כתובת שמתחילה ב־https:// ונגמרת ב־trycloudflare.com — זה הקישור לשמירה או לשליחה לטלפון.
  4. המחשב חייב להישאר דולק ומחובר לאינטרנט כל עוד רוצים שהקישור יעבוד.

הקישור משתנה בכל הרצה. אל תשתפי בפומבי — בזמן פעילות המנהרה כל מי שיש לו את הקישור יכול לגשת לנגן. מדריך טקסט: קישור-HTTPS-מהיר.txt בתיקייה.
