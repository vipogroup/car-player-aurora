# MOBILE QUICKSTART

1. פתחו במובייל: https://vipogroup.github.io/car-/car-player-standalone.html
2. התקינו למסך הבית:
   - Android: תפריט ⋮ -> התקן אפליקציה / הוסף למסך הבית
   - iPhone (Safari): שיתוף -> הוסף למסך הבית
3. כל משתמש עצמאי: שירים ופלייליסטים נשמרים במכשיר שלו בלבד (LocalStorage).
4. עדכונים:
   - האפליקציה בודקת גרסה חדשה אוטומטית (version.json + Service Worker).
   - אם יש עדכון: תופיע הודעה "עדכון חדש זמין".
   - לחיצה על "עדכן עכשיו" מושכת את הקבצים החדשים ומרעננת את האפליקציה.

הערה: זו הגרסה הסטטית. הגרסה המלאה עם unblocked_player.py דורשת שרת Python.
