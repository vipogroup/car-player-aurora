לינק ציבורי לכל העולם (אחרי הפעלת GitHub Pages):
https://vipogroup.github.io/car-/car-player-standalone.html
פתח את הנגן עכשיו
התקנה למסך הבית (כמו אפליקציה):
  • Android (Chrome/Edge): תפריט ⋮ → "התקן אפליקציה" / "הוסף למסך הבית"
  • iPhone (Safari): שיתוף → "הוסף למסך הבית"
עצמאות לכל משתמש:
כל מובייל שומר שירים + פלייליסטים בתוך המכשיר שלו. כלומר כל אחד רואה את הרשימה האישית שלו — נפרד מאחרים.
הגרסה הזו סטטית (PWA). המערכת המלאה unblocked_player.py עדיין דורשת שרת Python.
הורדת החבילה: car-music-player.zip
