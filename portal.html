אופציה 1: לינק ציבורי (הכי פשוט)
קישור: https://vipogroup.github.io/car-/car-player-standalone.html
פתח את הנגן הציבורי

זה דפדפן/PWA סטטי (לא שרת Python מקומי).

אופציה 2: שרת מקומי במחשב שלך + חיבור מהמובייל
  1. הורד חבילת התקנה: local-server-unblocked.zip
  2. חלץ לקובץ במחשב.
  3. התקן Python ואז pip install -r requirements.txt.
  4. הרץ start-server-lan.bat.
  5. במובייל (אותה Wi-Fi) פתח: http://IP-של-המחשב:5600.
מדריך מפורט: LOCAL-SERVER-SETUP.md
התקנה למסך הבית:
  • Android: תפריט ⋮ -> "התקן אפליקציה" / "הוסף למסך הבית"
  • iPhone: שיתוף -> "הוסף למסך הבית"
